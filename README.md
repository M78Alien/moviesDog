# moviesDog 影趣购电影票务平台

影趣购项目是作为毕业设计的项目，主要分为后台管理、小程序客户端和后端服务器。

项目地址： https://github.com/M78Alien/moviesDog
## 项目目录

- moviesDogApp： 微信小程序客户端源码
- moviesDogWeb： 后台管理系统源码
- moviesDogServer： 后端服务器源码
- sqlData： 数据库文件

## 相关技术

- **前端**：JavaScript、Vue、ElementUI、微信小程序、Vant Weapp
- **后端**：Springboot、MySQL、
- **云服务**：阿里云OSS、阿里云ECS
